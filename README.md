# 🔋 AI-Based EV Battery Health Prediction & Failure Detection

## 📌 Project Overview
This project uses Machine Learning to predict the health and safety of Electric Vehicle (EV) batteries.  
It analyzes battery parameters and provides insights such as SOH (State of Health), failure probability, and safety recommendations.

---

## 🚀 Features
- 🔍 Battery Health Prediction (SOH %)
- ⚠️ Failure Probability Detection
- 📊 Risk Level Classification (Low / Medium / High)
- 🌡️ Temperature Safety Analysis
- 📈 Interactive Graphs using Gradio

---

## 🧠 Input Parameters
The model takes the following inputs:

- SOC (State of Charge)
- Voltage
- Current
- Temperature
- Charge Cycles
- Internal Resistance
- Capacity

---

## 📤 Output Predictions
The system predicts:

- ✅ Battery Health (SOH %)
- ⚠️ Battery Failure Probability
- 🚨 Risk Level (Low / Medium / High)
- 🌡️ Temperature Condition & Safety Advice

---

## 📊 Visualization (Gradio UI)
The project also provides interactive visualizations:

- SOH degradation vs Charge Cycles
- Temperature vs SOH
- Failure Distribution
- Temperature Analysis Graphs

---

## 🛠️ Technologies Used
- Python
- Machine Learning (Scikit-learn)
- Pandas, NumPy, Matplotlib
